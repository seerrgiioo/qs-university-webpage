A Step-by-Step Guide to Implement a Simple NUI (HTML UI) in FiveM

This guide will show you how to set up a basic NUI (Native User Interface) for your FiveM server.
You'll learn how to connect Lua client commands with HTML UI, JavaScript logic, and CSS styling.

1.1 Client-Side Command (Lua)

The RegisterCommand> in Lua creates a command that can be executed in-game. Today we will use this command to open the NUI.

RegisterCommand('OpenNui', function()
    SendNUIMessage({
        action = "open"
    })
end)

What it Does:

  • RegisterCommand: Creates a custom command /OpenNui that players can use.
  • SendNUIMessage: Sends data to the HTML interface. Here, it sends an action value "open" to tell the UI to appear.

How it Works: When a player types /OpenNui in the game chat, the SendNUIMessage triggers the NUI to open.

Here you can see the SendNUIMessage function in the FiveM Documentation.

1.2 Creating the NUI (HTML UI)

Now that we have the command, we need to create the NUI.

Open the index.html at script/NUI/ and create the NUI with the code below:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style/nui.css">
    <title>Quasar University | NUI </title>
</head>
<body>
    <div id="main-container">
        <button id="open-nui">Open NUI</button>

        <div id="nui-container">
            <h1>Quasar University</h1>
            <p><u>Welcome to Quasar University</u></p>
            <button id="close-nui">Close NUI</button>
        </div>
    </div>
    <script src="script/nui.js"></script>
</body>
</html>

This code sets up the basic structure of the NUI. The index.html file contains the HTML structure, the script/nui.js file contains the JavaScript logic, and the style/nui.css file contains the CSS styling.

1.3 JavaScript Linking (Client-Side)

To link the JavaScript file to the HTML, you need to add a <script> tag in the index.html file.

<script src="script/nui.js"></script>

What it Does:

  • src="script/nui.js": Specifies the path to the JavaScript file.

How it Works: The JavaScript file is linked to the HTML, making it accessible for the client-side code to use.

Now you have the JavaScript file linked to the HTML, you can start working on the JavaScript code.

We receive the data from the SendNUIMessage using the window.addEventListener('message', (event) => {

Here will see the code at JS:

document.addEventListener('DOMContentLoaded', () => {

    window.addEventListener('message', (event) => {
        const data = event.data;
        if (typeof data !== 'object' || !data.action) return;
    
        
        switch (data.action) {
            case 'openNUI':
                const openNUIButton = document.getElementById('open-nui');
                openNUIButton.style.display = 'block';
                break;
        }
    });
    
    // Get the elements
    const openNUIButton = document.getElementById('open-nui');
    const closeNUIButton = document.getElementById('close-nui');
    const nuiContainer = document.getElementById('nui-container');

    // Function to open the NUI
    openNUIButton.addEventListener('click', () => {
        nuiContainer.style.display = 'block';
    });

    // Function to close the NUI
    closeNUIButton.addEventListener('click', () => {
        nuiContainer.style.display = 'none';
        openNUIButton.style.display = 'none';
        fetch(`https://${GetParentResourceName()}/close`, { method: 'POST' });
    });
});

What it Does:

document.addEventListener('DOMContentLoaded', () => {: Listens for the DOM content to be loaded.
window.addEventListener('message', (event) => {: Listens for messages from the client.

const data = event.data;: Retrieves the data sent from the client.

if (typeof data !== 'object' || !data.type) return;: Checks if the data is an object and has a type property.

switch (data.type) {: Checks the type of the data.

case 'openNUI':: Handles the 'openNUI' case.

openNUIButton.style.display = 'block';: Shows the open NUI button.

break;: Breaks out of the switch statement.
