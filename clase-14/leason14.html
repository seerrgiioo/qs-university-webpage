1. Introduction

Conditions in Lua allow you to make decisions in your code based on specific criteria. This lesson covers how to use `if`, `else`, `elseif`, and logical operators to manage different scenarios effectively.

2. Basic `if` Statement

The `if` statement evaluates whether a condition is true.


    local age = 18
    
    if age >= 18 then
        print("You are an adult.")
    end

Explanation:

  • The condition age >= 18 checks if the value of age is 18 or higher.
  • If the condition is true, the block inside then will execute.

3. `if` with `else`


    local age = 16
    
    if age >= 18 then
        print("You are an adult.")
    else
        print("You are a minor.")
    end

Explanation:

  • If the condition age >= 18 is false, the block under else will execute.

4. `if` with `elseif`


    local score = 85
    
    if score >= 90 then
        print("Your grade is excellent.")
    elseif score >= 75 then
        print("Your grade is good.")
    else
        print("You need to improve.")
    end

Explanation:

  • When the first condition score >= 90 is false, the code checks the elseif condition score >= 75.
  • If both conditions are false, the else block executes.

5. Logical Operators


    local isStudent = true
    local hasDiscount = true
    
    if isStudent and hasDiscount then
        print("You qualify for a special discount.")
    end

Explanation:

  • and: Both conditions must be true for the block to execute.
  • or: At least one condition must be true.
  • not: Inverts the logical value of a condition.

6. Truthy and Falsy Values


    if 0 then
        print("0 is considered true in Lua.")
    end
    
    if nil then
        print("This will not execute because nil is false.")
    end

Explanation:

  • In Lua, only false and nil are considered falsy.
  • All other values (e.g., 0, "", true) are truthy.

7. Nested Conditions


    local age = 25
    local hasLicense = true
    
    if age >= 18 then
        if hasLicense then
            print("You can drive.")
        else
            print("You need a license to drive.")
        end
    else
        print("You must be an adult to drive.")
    end

Explanation:

  • The outer if checks whether the person is an adult.
  • The inner if checks if the person has a license.

8. Conclusion

Mastering conditional statements in Lua is essential for creating dynamic and flexible code. Experiment with different scenarios and practice using `if`, `else`, and logical operators in your programs.
